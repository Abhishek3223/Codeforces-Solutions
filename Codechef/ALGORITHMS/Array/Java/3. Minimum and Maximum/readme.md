## Problem
Given an array **arr** of size **n** of integers. <br> Your task is to find the **minimum and maximum** elements in the array.

### Example 1
```
Input:
n = 6
arr[] = {3, 2, 1, 56, 10000, 167}
Output:
min = 1, max =  10000
```
### Example 2
```
Input:
n = 5
arr[]  = {1, 345, 234, 21, 56789}
Output:
min = 1, max = 56789
```
