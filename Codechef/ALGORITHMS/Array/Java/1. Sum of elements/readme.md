## Problem
Given an array **arr** of size **n** (*integers*) <br> Your task is to find and return the **sum** of all elements in the array.

### Example 1
```
Input:
n = 6
arr[] = {3, 10, 1, 56, 2, 25}
Output:
sum = 97
```
### Example 2
```
Input:
n = 5
arr[]  = {1, 3, -2, 20, -5}
Output:
sum = 17
```
